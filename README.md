# 🚀 Machine Learning & Data Science Projects Portfolio

Welcome to my machine learning repository! This portfolio is a structured, hands-on record of my journey through data science. My core learning philosophy is centered on **building from the ground up**: mastering fundamental machine learning algorithms through low-level math and custom implementations before transitioning to advanced, production-ready pipelines using modern industrial frameworks.

---

## 🗺️ My Learning Roadmap & Core Philosophy

I believe that the best data scientists don't just treat machine learning models like a "black box." Instead of simply importing libraries blindly, my path is divided into progressive phases:

1. **Foundations from Scratch:** Writing optimization routines (like Gradient Descent) in pure Python and NumPy to understand the underlying calculus and linear algebra.
2. **Supervised Learning & Feature Engineering:** Leveraging standard frameworks (`Scikit-Learn`, `Pandas`) to manage real-world datasets, handle scaling, and solve classification/regression problems.
3. **Advanced Architecture (Upcoming):** Transitioning into tree-based ensemble methods (Random Forests, XGBoost), unsupervised learning frameworks (Clustering, PCA), and deep learning.
